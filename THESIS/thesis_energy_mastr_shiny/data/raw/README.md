# `raw/` — originals & bulk downloads

Put **verbatim downloads** here (PDF, xlsx, zip, API JSON snapshots) while you work.

## Rules

- **Large files** should stay **local** or in object storage; this folder is **git-ignored by default** (only this README and `.gitignore` are tracked).
- If you must commit a small extract (for example under ~1 MB CSV under licence), add an exception in **`raw/.gitignore`** or move it to **`../processed/`** after cleaning.
- For each batch, add a one-line entry below: **date — source — filename — licence note**.
